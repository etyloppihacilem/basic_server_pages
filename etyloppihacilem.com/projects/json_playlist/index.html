What are we talking about

Take a look at this page (not responsive yet, use a desktop).

Why did I built this

I was willing to use a json array of deezer id to make this page work, as I am using the Deezer Api to recover album artwork and 30s sample of songs. Thats is why I built a whole tool to create and edit thoses playlists.

Non-exhaustive list of features

  • Search for songs on deezer database
  • Click on album name or artist name to search it automatically
  • Select what metadata to save in json
  • Load and safe to file
  • Autoselect metadata depending on loaded file
  • Listen to song when clicking on artwork (this also works on the discover page)
